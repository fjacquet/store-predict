**4. Create `src/store_predict/ui/pages/review.py`**:

```python
@ui.page("/review")
async def review_page() -> None:
```

Page logic:

1. Load data: `df = load_session_data()`. If None, show "No data uploaded" message with a link to /upload, return early.
2. Load DRR table: `drr_table = DRRTable.from_csv(DRR_CSV_PATH)`
3. Get workload options: Use `get_workload_options()` from state module
4. Prepare row_data: `row_data = df.to_dict(orient="records")`
5. Get category list for dropdown: `categories = drr_table.categories`

Layout (inside `with layout("StorePredict - Review"), ui.column().classes("w-full max-w-7xl mx-auto p-4 gap-4")`):

- Title row: "Review Classifications" (text-2xl font-bold) + project name display from `get_project_name()`
- Summary stats container: Use a `ui.column` with an id/ref to hold stats. Call `build_summary_stats(row_data)` initially. This container will be rebuilt on changes.
- AG Grid table: `grid = create_vm_table(row_data, categories, on_cell_changed=handle_cell_change, on_row_clicked=handle_row_click)`

**handle_cell_change(e) callback:**

- Extract changed row data from `e.args`
- The AG Grid `cellValueChanged` event provides `e.args` with keys: `data` (full row), `colId` (changed column), `newValue`, `oldValue`
- If `colId == "workload_category"`:
  - Look up first matching subcategory for the new category from drr_table entries
  - Look up new DRR: `drr_table.get_ratio(new_category, subcategory)`
  - Update the row in row_data list (find by vm_name match)
  - Update `row["workload_category"]`, `row["workload_subcategory"]`, `row["drr"]`
  - Call `grid.options["rowData"] = row_data` then `grid.update()`
  - Rebuild summary stats (clear container, call build_summary_stats again)
  - Save updated data: `save_session_data(pd.DataFrame(row_data), get_project_name())`

**handle_row_click(e) async callback:**

- Extract `vm_name` from `e.args["data"]["vm_name"]`
- Get current workload as list: `[e.args["data"]["workload_category"]]`
- Build options list from workload_options (labels)
- Show dialog: `result = await WorkloadDialog(vm_name, current, options_list)`
- If result is not None and len(result) > 0:
  - Build workload tuples from selected labels for DRR lookup
  - Calculate conservative DRR: `drr_table.get_conservative_ratio(workload_tuples)`
  - Update row: set `workload_category` to first selected (or comma-joined), `drr` to conservative value
  - Update grid, rebuild stats, save session data (same pattern as cell change)

**5. Update `src/store_predict/main.py`**:
