Create `src/store_predict/pipeline/calculation.py` with ZERO UI imports (NFR-2.4).

Implement these dataclasses (all frozen=True):
- `VMCalculation`: vm_name, workload_category, provisioned_mib, in_use_mib, drr, required_mib
- `WorkloadGroupResult`: category, vm_count, total_provisioned_mib, total_in_use_mib, avg_drr, total_required_mib
- `CalculationSummary`: vm_calculations (list), workload_groups (list), total_vms, total_provisioned_mib, total_in_use_mib, total_required_mib, weighted_avg_drr

Implement `calculate(row_data: list[dict[str, Any]]) -> CalculationSummary`:
1. Early return for empty list: all zeros, empty lists
2. For each row dict, compute per-VM result:
   - Use `.get(key, default)` for all field access (provisioned_mib=0, in_use_mib=0, drr=5.0, vm_name="", workload_category="Unknown (Reducible)")
   - Guard DRR: `drr = max(float(row.get("drr", 5.0)), 0.1)`
   - `required_mib = provisioned_mib / drr`
3. Group VMCalculation results by workload_category using defaultdict(list)
4. For each group: compute vm_count, total_provisioned, total_in_use, total_required, avg_drr (= group_provisioned / group_required)
5. Sort groups by category name
6. Compute grand totals: sum of all provisioned, in_use, required
7. Weighted avg DRR: total_provisioned / total_required (guard: if total_required == 0, return 0.0)

Export all public symbols in `__all__`.

Update `src/store_predict/pipeline/__init__.py` to export calculation symbols.
