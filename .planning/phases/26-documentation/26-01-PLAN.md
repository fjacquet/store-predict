# Plan 26-01: Update PRD to v5.0

## Objective

Update `docs/prd.md` from v4.0 to v5.0 to reflect features delivered in Phases 23–25 (multi-cluster compute breakdown, health findings export, configurable vMSC/DR site split ratios). DOCS-01 requires a living PRD covering tool scope, user personas, use cases, feature rationale, and non-functional requirements.

The PRD already exists and is comprehensive. This plan adds the v5.0 feature additions without reworking existing content.

## Context

- PRD current version: 4.0 (as of 2026-02-22)
- New features to document (Phases 23–25):
  - **Phase 23**: Per-cluster compute breakdown table on /compute; health findings scoped per cluster
  - **Phase 24**: Health findings exported to PDF (summary + detail appendix) and Excel (Findings worksheet)
  - **Phase 25**: Configurable vMSC split ratio (e.g., 60/40) and A/P DR active percentage; per-site host count display on /compute
