**Change 2 — Replace global `_check_hw_version()` with `_check_hw_version_per_cluster()`:**
Remove the existing `_check_hw_version(active)` call in `run_health_checks()` and replace with `_check_hw_version_per_cluster(active)`.

Implement `_check_hw_version_per_cluster(df: pd.DataFrame) -> list[HealthFinding]`:
- Same sentinel guard: if `(hw > 0).sum() == 0`, return `[]`.
- Normalize cluster column: `df["cluster"].fillna("").astype(str).str.strip().replace("", "(No Cluster)")` — this is just for groupby labeling; do NOT log it.
- For each `cluster_name, group` in `df.groupby(cluster_col, sort=True)`:
  - Compute `group_hw` from `group.get("hw_version", ...)` same as existing check.
  - If `(group_hw > 0).sum() == 0`, skip (no HW data for this cluster).
  - Check `very_old_mask = (group_hw > 0) & (group_hw < _VERY_OLD_HW_VERSION)`.
  - If any: emit `HealthFinding(check_id="best_practice.very_old_hw_version", severity=Severity.CRITICAL, title="health.very_old_hw_version.title", detail="health.very_old_hw_version.detail", affected_count=len(group[very_old_mask]), affected_vms=tuple(group[very_old_mask]["vm_name"].head(5).tolist()), cluster=str(cluster_name))`.
  - Else: check `old_mask = (group_hw > 0) & (group_hw >= _VERY_OLD_HW_VERSION) & (group_hw < _OLD_HW_VERSION)`.
  - If any: emit `HealthFinding(check_id="best_practice.old_hw_version", ..., cluster=str(cluster_name))` with same titles/details as before.
- Return `findings` list (one finding per affected cluster).

**Change 3 — Add `_check_small_cluster_ha()` for CLUS-04 HA ratio check:**
Add new function `_check_small_cluster_ha(df: pd.DataFrame) -> list[HealthFinding]`:
- Normalize cluster column same as above.
- For each cluster: if `len(group) < 3`, emit `HealthFinding(check_id="best_practice.small_cluster_ha", severity=Severity.WARNING, title="health.small_cluster_ha.title", detail="health.small_cluster_ha.detail", affected_count=len(group), affected_vms=tuple(group["vm_name"].head(5).tolist()), cluster=str(cluster_name))`.
- Skip if cluster is "(No Cluster)" — standalone hosts don't have HA context.
- Return findings.

Add `findings.extend(_check_small_cluster_ha(active))` in `run_health_checks()` AFTER the `_check_no_cluster` call.

The old `_check_hw_version()` function body can remain as a private helper but should not be called from `run_health_checks()` anymore — mark it with a comment "# Superseded by _check_hw_version_per_cluster()" or delete it entirely.

Do NOT log cluster names or VM names.
