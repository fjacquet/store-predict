**Private helpers:**

`_clamp_ratio(ratio: float) -> float`:

- Returns `max(0.5, min(20.0, ratio))`

`_hosts_n1(total_vcpus: int, host_pcores: int, ratio: float) -> int`:

- Guard: if any argument <= 0, return 0
- `capacity = max(1, host_pcores) * max(0.5, ratio)`
- `return math.ceil(total_vcpus / capacity) + 1` for N+1 HA

`_hosts_by_ram(total_ram_gib: float, host_ram_gib: int) -> int`:

- Guard: if either <= 0, return 0
- `return math.ceil(total_ram_gib / max(1, host_ram_gib)) + 1` for N+1 HA

`_vmsc_sites(df: pd.DataFrame) -> list[str]`:

- If "datacenter" not in df.columns, return []
- Return `[v for v in df["datacenter"].dropna().unique() if str(v).strip()]`

`_empty_result(host_config: HostConfig, overcommit_ratio: float) -> ComputeSizingResult`:

- Returns zero-count result with `has_data=False`, all host counts 0, empty vmsc_sites tuple

**Entry point `compute_sizing()`:**

```python
def compute_sizing(
    df: pd.DataFrame | None,
    host_config: HostConfig,
    overcommit_ratio: float = 4.0,
    vmsc_enabled: bool = False,
    ap_enabled: bool = False,
) -> ComputeSizingResult:
```

Implementation steps:

1. Return `_empty_result(host_config, overcommit_ratio)` if df is None or df.empty
2. Clamp ratio: `ratio = _clamp_ratio(overcommit_ratio)`
3. Filter: `active = df[(df["is_powered_on"] == True) & (df["is_template"] == False)]` (use noqa: E712)
4. `excluded_count = len(df) - len(active)`
5. If active is empty: return `_empty_result` with `excluded_vm_count=excluded_count`
6. Aggregate using `pd.to_numeric(..., errors="coerce").fillna(0)` for both `num_cpus` and `memory_mib` (session round-trip may produce None/object dtype)
7. `total_vcpus = int(pd.to_numeric(active["num_cpus"], errors="coerce").fillna(0).sum())`
8. `total_ram_mib = float(pd.to_numeric(active["memory_mib"], errors="coerce").fillna(0).sum())`
9. `total_ram_gib = total_ram_mib / 1024.0`
10. If total_vcpus == 0 AND total_ram_gib == 0.0: return `_empty_result` with excluded count
11. `host_pcores = host_config.total_cores`
12. `hv = _hosts_n1(total_vcpus, host_pcores, ratio)`
13. `hr = _hosts_by_ram(total_ram_gib, host_config.ram_gib)`
14. `hosts_n1 = max(hv, hr)`
15. vMSC: `sites = _vmsc_sites(active)`, `vmsc_avail = len(sites) >= 2`
16. `vmsc_hosts = _hosts_n1(total_vcpus, host_pcores, ratio) if (vmsc_enabled and vmsc_avail) else 0`
17. Active/Passive: `ap_primary = hosts_n1`, `ap_secondary = max(1, math.ceil(hosts_n1 / 2))`
18. Return fully-populated `ComputeSizingResult(...)`
