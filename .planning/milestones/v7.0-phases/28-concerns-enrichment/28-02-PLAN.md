    **CSV generator (`generate_concerns_csv`):**

    ```python
    def generate_concerns_csv(health_result: HealthCheckResult) -> bytes:
    ```

    - Use `import csv` and `io.StringIO`
    - Write header row: `severity,check_id,title,detail,remediation,affected_count,cluster`
    - For each finding in `health_result.findings`:
      - `severity`: `finding.severity.value` (e.g. "critical")
      - `check_id`: `finding.check_id`
      - `title`: `finding.title` (the i18n key string, e.g. "health.missing_os.title")
      - `detail`: `finding.detail` (the i18n key string)
      - `remediation`: `finding.remediation`
      - `affected_count`: `str(finding.affected_count)`
      - `cluster`: `finding.cluster`
    - Return `buf.getvalue().encode("utf-8-sig")` (UTF-8 with BOM for Excel compatibility)

    **PDF generator (`generate_concerns_pdf`):**

    ```python
    def generate_concerns_pdf(
        health_result: HealthCheckResult,
        project_name: str = "",
        locale: str = "fr",
    ) -> bytes:
    ```

    - Use `io.BytesIO`, `reportlab.platypus.SimpleDocTemplate`, A4 page size
    - Register Vera + VeraBd fonts (same pattern as `pdf_report.py` — import `reportlab`, use `os.path.join(os.path.dirname(reportlab.__file__), "fonts")`)
    - Build flowables list:
      1. **Header**: Bold title "Health Check Concerns Report", project name if provided, generation date (`datetime.now(UTC).strftime("%Y-%m-%d")`)
      2. **Summary bar**: Three `Paragraph` elements — "{N} Critical", "{N} Warning", "{N} Info" — colored red/yellow/blue using `colors.HexColor`
      3. **Severity legend**: Small italic paragraph explaining the severity levels
      4. **Spacer**
      5. **For each finding**: One `Table` per finding with 2 rows:
         - Row 1: severity badge cell (colored background) + title text cell
         - Row 2 (colspan): detail text + remediation hint italic
         - Apply `TableStyle` with colored left border (red for CRITICAL, `#f59e0b` for WARNING, `#3b82f6` for INFO)
         - If `finding.affected_count > 0`, add affected count below
    - Severity color map: `CRITICAL → #dc2626`, `WARNING → #f59e0b`, `INFO → #3b82f6`
    - Use `Vera` font throughout (not the system default — required for French locale support)
    - Return `buf.getvalue()` (bytes)

    **Exports:**
    ```python

    ```

    Note: Do NOT import from `store_predict.ui` — this is a pure service module with zero UI imports (same pattern as `health_checks.py`).
