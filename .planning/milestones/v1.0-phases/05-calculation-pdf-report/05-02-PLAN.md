  <name>Task 1: Build PDF report generator service</name>
  <files>src/store_predict/services/pdf_report.py</files>
  <action>
Create `src/store_predict/services/pdf_report.py` with `generate_report_pdf(summary: CalculationSummary, project_name: str) -> bytes`.

Implementation:

1. **Font registration** — Register Vera and VeraBd TTF fonts from ReportLab's bundled font directory:

   ```python
   import reportlab
   font_dir = os.path.join(os.path.dirname(reportlab.__file__), 'fonts')
   pdfmetrics.registerFont(TTFont('Vera', os.path.join(font_dir, 'Vera.ttf')))
   pdfmetrics.registerFont(TTFont('VeraBd', os.path.join(font_dir, 'VeraBd.ttf')))
   ```

2. **Document setup** — Use `SimpleDocTemplate` with A4 page, 20mm margins, BytesIO buffer. Title: "StorePredict Report - {project_name}".

3. **Header callback (`_draw_header`)** — onFirstPage callback that draws:
   - Dark blue rectangle bar (0, height-50 to width, 50) as brand element
   - White "StorePredict Sizing Report" title in VeraBd 18pt
   - Project name and date below the bar in Vera 11pt

4. **Summary section** — Build flowables for summary cards:
   - Project Name, Date, Total VMs, Total Provisioned (GiB), Total In Use (GiB), Weighted Avg DRR, Required Capacity (GiB)
   - Use `Paragraph` with custom styles based on `getSampleStyleSheet()`
   - Format storage values: MiB / 1024 → GiB with 1 decimal. If GiB > 1024, also show TiB in parentheses.

5. **Workload breakdown table** — ReportLab `Table` with:
   - Header row: Category, VMs, Provisioned (GiB), Avg DRR, Required (GiB)
   - One row per WorkloadGroupResult from summary.workload_groups
   - Totals row at bottom (bold)
   - TableStyle: dark blue header (#1e3a5f), white text, alternating row backgrounds, grey grid, bold totals row
   - Column widths: [180, 50, 100, 70, 100] — should fit A4 with 20mm margins
   - Font size 9pt for table body to fit on one page

6. **Build document** — `doc.build(story, onFirstPage=_draw_header)` then `buffer.getvalue()`.

7. **Helper: `format_storage(mib: float) -> str`** — Converts MiB to GiB string. If >= 1024 GiB, append TiB. E.g., "1536.0 GiB (1.5 TiB)".

8. **Helper: `sanitize_filename(name: str) -> str`** — Replace non-alphanumeric chars with underscore.

Export: `generate_report_pdf`, `sanitize_filename`, `format_storage` in `__all__`.

Run: `rtk ruff check src/store_predict/services/pdf_report.py` and `rtk mypy src/store_predict/services/pdf_report.py` — clean.
