Create `src/store_predict/pipeline/parsers/rvtools.py`:
- Implement `parse_rvtools(path: Path) -> pd.DataFrame`:
  1. Read only the "vInfo" sheet: `pd.read_excel(path, sheet_name="vInfo", engine="openpyxl")`. Wrap in try/except to catch KeyError (missing sheet) or ValueError, re-raise as IngestionError.
  2. Strip column whitespace: `df.columns = df.columns.str.strip()`
  3. Call `resolve_columns(df, RVTOOLS_ALIASES, REQUIRED_RVTOOLS_COLUMNS)` to get col_map.
  4. Build result DataFrame with canonical column names:
     - vm_name: from col_map["vm_name"], fillna("")
     - os_name: from col_map["os_name"], fillna("")
     - provisioned_mib: pd.to_numeric with errors="coerce", fillna(0.0)
     - in_use_mib: pd.to_numeric with errors="coerce", fillna(0.0)
     - datacenter: from col_map["datacenter"] if found, else "" for all rows
     - cluster: from col_map["cluster"] if found, else "" for all rows
     - is_template: CRITICAL — use `.fillna(False).astype(bool)` to handle NaN (research pitfall 1)
     - is_powered_on: from powerstate column if found, `str.lower() == "poweredon"`, else True for all rows
     - source_format: FileFormat.RVTOOLS.value for all rows
  5. Return the DataFrame with only CANONICAL_COLUMNS.
  - Do NOT apply unit conversion — RVTools "MB" values are already MiB (research pitfall 2).
  - Do NOT filter templates here — that happens in the orchestrator.

  5. Build result DataFrame with source_format=FileFormat.LIVEOPTICS_CSV.value.
  6. Return canonical DataFrame.
