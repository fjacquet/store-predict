Create `src/store_predict/pipeline/ingestion.py`:

Implement `detect_format(path: Path) -> FileFormat`:

1. Validate file exists — raise IngestionError if not.
2. Check extension (path.suffix.lower()):
   - ".csv": Read header row only with `pd.read_csv(path, nrows=0)`, strip column whitespace. Check for LiveOptics signature columns ("VM Name" or "VM OS"). Return FileFormat.LIVEOPTICS_CSV if found. Raise IngestionError listing expected columns if not found.
   - ".xlsx": Open with `openpyxl.load_workbook(path, read_only=True)`, get sheet names, close workbook. If "vInfo" in sheets -> FileFormat.RVTOOLS. If "VMs" in sheets -> FileFormat.LIVEOPTICS_XLSX. Raise IngestionError listing first 5 sheet names if neither found.
   - Other extensions: Raise IngestionError("Unsupported file type: {suffix}. Please upload .xlsx (RVTools or LiveOptics) or .csv (LiveOptics).").

Implement `ingest_file(path: Path) -> pd.DataFrame`:

1. Call detect_format(path) to get format.
2. Dispatch to appropriate parser:
   - FileFormat.RVTOOLS -> parse_rvtools(path)
   - FileFormat.LIVEOPTICS_XLSX -> parse_liveoptics_xlsx(path)
   - FileFormat.LIVEOPTICS_CSV -> parse_liveoptics_csv(path)
3. Filter out template VMs: `df = df[~df["is_template"]].reset_index(drop=True)`
4. Return the filtered DataFrame.

Update `src/store_predict/pipeline/__init__.py`:

- Re-export: detect_format, ingest_file, IngestionError, FileFormat.

- Include: 2 powered-on VMs, 1 powered-off VM, 1 template VM, various OS types.
- Use realistic VM names (e.g., "WEBSRV01", "SQLDB02", "DVDI-PC01", "TMPL-WIN10").
- Use realistic MiB values (e.g., 61440, 32978, 102400).
