<task type="auto">
  <name>Task 2: Implement upload page with pipeline integration</name>
  <files>src/store_predict/ui/pages/upload.py</files>
  <action>
Replace the placeholder upload page with a full implementation. Use the existing `layout()` context manager and follow the combined with-statement pattern (`with layout(), ui.column()`).

Page structure at `@ui.page("/upload")`:

1. Title: "Upload Workload Data" (text-3xl font-bold)
2. Project name input: `ui.input` with label "Project Name", placeholder "e.g., Customer-DC-Migration-2026", bound to `app.storage.tab` via `on_change` callback that calls `set_project_name(e.value)`. Set initial value from `get_project_name()`.
3. File upload dropzone: `ui.upload` with:
   - `label="Drop RVTools or LiveOptics file here (.xlsx, .csv)"`
   - `on_upload=handle_upload` (async handler)
   - `auto_upload=True`
   - `max_file_size=50_000_000` (50MB)
   - `.props('accept=".xlsx,.csv"')`
   - `.classes("w-full")`
4. Status area: A `ui.label` that shows upload progress/errors, initially hidden.

`handle_upload` async function:

1. Write uploaded content to a temp file using `tempfile.NamedTemporaryFile(suffix=Path(e.name).suffix, delete=False)`
2. Read content via `e.content.read()`
3. Call `ingest_file(tmp_path)` to parse and normalize
4. Create `RuleRegistry(build_default_rules())` and call `classify_dataframe(df, registry)`
5. Load DRR table via `DRRTable.from_csv(DRR_CSV_PATH)` and add `drr` column: `df["drr"] = df.apply(lambda r: drr_table.get_ratio(r["workload_category"], r["workload_subcategory"]), axis=1)`
6. Call `save_session_data(df, get_project_name())`
7. Show success notification: `ui.notify(f"Loaded {len(df)} VMs", type="positive")`
8. Navigate to review: `ui.navigate.to("/review")`
9. Wrap in try/except: catch `IngestionError` and general `Exception`, show `ui.notify(str(exc), type="negative")`
10. `finally:` delete temp file via `tmp_path.unlink(missing_ok=True)`

Imports needed:

- `tempfile`, `from pathlib import Path`
- `from nicegui import app, ui`

- `from store_predict.pipeline.errors import IngestionError`

Use Tailwind classes for styling: centered column (`max-w-2xl mx-auto p-8 gap-6`), card wrapper for the upload area.
