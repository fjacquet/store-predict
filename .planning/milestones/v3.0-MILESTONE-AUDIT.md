## Remaining Tech Debt (Non-Blocking)

### `generate_report_pdf()` — Dead Code at Runtime

`pdf_report.py` exports `generate_report_pdf()` in `__all__` but it is never called from the running application. The Playwright path (`report_print.py`) is the only active PDF backend. The ReportLab function is maintained and tested (12+ tests exercise it) but constitutes dead code in production.

**Severity:** Low — maintenance concern only. No user-facing impact.
**Recommendation:** Either remove from `__all__` and add a docstring marking it as test-only, or restore a runtime call path for non-Playwright environments.
