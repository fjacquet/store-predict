## Configuration

**Environment:**
- `STORAGE_SECRET` - NiceGUI session encryption key (env var, defaults to `"dev-only-not-for-production"`)
- `LLM_ENABLED` - Enable/disable LLM classification fallback (default: `false`)
- `LLM_MODEL` - LLM model identifier for OpenRouter (default: `mistralai/mistral-small-3.1-24b-instruct`)
- `LLM_API_KEY` - OpenRouter API key (must be set if `LLM_ENABLED=true`)
- `LLM_API_BASE` - Custom LLM API endpoint (default: `https://openrouter.ai/api/v1`)
- `LLM_TIMEOUT` - Per-request timeout in seconds (default: `30`)
- `LLM_MAX_CONCURRENT` - Max simultaneous LLM calls (default: `5`)
- `LLM_BATCH_SIZE` - VMs per LLM request (default: `10`)

**Build:**
- `pyproject.toml` - Central configuration (dependencies, metadata, build backend, tool configs)
- `pyrightconfig.json` - Pyright type checking (venv at `.venv`, Python 3.14, standard mode)
- `mkdocs.yml` - Documentation site configuration

**Type Checking:**
- mypy configured as `strict = true` in `pyproject.toml`
- Overrides for untyped stubs: `nicegui`, `reportlab`, `i18n`, `xlsxwriter`, `litellm`, `matplotlib`, `playwright`
- Coverage omit: `src/store_predict/ui/`, `src/store_predict/main.py`, `src/store_predict/logging_config.py`

## Platform Requirements

**Development:**
- Python 3.12+ installed locally
- `.venv/` virtual environment (created via `uv venv .venv`)
- uv binary available (`pip install uv`)
- Operating System: macOS, Linux, Windows (Bash-compatible shell for CI)

**Production:**
- Docker (single-container deployment via `Dockerfile`)
- Python 3.12-slim base image (`FROM python:3.12-slim`)
- Chromium + dependencies installed in container (`playwright install chromium --with-deps`)
- Port 8080 exposed (NiceGUI default)
- HEALTHCHECK directive on port 8080 (HTTP GET check)
- Non-root user `appuser` for security

**CI/CD:**
- GitHub Actions - Two workflows:
  - `ci.yml` - Lint, type-check, test, upload coverage to Codecov (pushes to `main` branch)
  - `docs.yml` - Deploy MkDocs to GitHub Pages on version tag (`v*`)
